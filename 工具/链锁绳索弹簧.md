# 1.Distance Joint 2D
用于控制两物体间的间距  

但是并不具有弹性，如果场景中物体相对于Rb2D的真实距离不等于

组件中  距离  参数，那么两边都会受到强制复原 

适合作为陷阱的摆锤 或者 布娃娃系统的武器
![[Pasted image 20251226165332.png]]
# 2.Spring Joint 2D
具有弹性的杆子  弹力会强制两个目标保持一定距离  

如果想要实现真正弹簧绳索的效果  需要在真实距离小于 组件中 距离时  
手动关闭脚本  并且适应性的设置DampingRatio于频率 

适合作为弹簧 弹绳
![[Pasted image 20251226165353.png]]![[Pasted image 20251226165356.png]]
# 3.Hinge Joint 2D  
不具有弹性的连接器

相比Distance Joint 2D对于距离的限制更温柔

可以限制活动角与死角

可以使用Motor来转动

适合于锁链  门的扭矩  关节
![[Pasted image 20251226165409.png]]![[Pasted image 20251226165411.png]]
# 4.Wheel Joint 2D
轮子连接器

将rb连接在目标身上后 还需要手动修改   已连接锚点的位置  保证固定在目标的车轮轴上

修改Motor可以获得转动速度
